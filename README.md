# Dr. Shreyas S R - Personal Website

Static personal academic website designed for GitHub Pages.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Open the repository settings on GitHub.
3. Go to **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the repository root.

The site entry point is `index.html`, so it works directly with GitHub Pages without a build step.
